# Spotify Clone

A Spotify mobile app UI clone.

## Made with

* [Expo](https://expo.io/)
* [LinearGradient](https://docs.expo.io/versions/latest/sdk/linear-gradient/)
* [React Native](https://reactnative.dev/)

* [Visual Studio Code](https://code.visualstudio.com/)

```bash
# Install depedencies
npm install

# Run the app
npm start
```
